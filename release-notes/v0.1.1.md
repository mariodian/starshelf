# v0.1.1

Starshelf is now available for Firefox, and the release pipeline builds both Chrome and Firefox extensions in one shot.

## Highlights

- **Firefox support** — Starshelf can now be installed and used in Firefox, with a dedicated build and packaging setup.
- **Streamlined releases** — Both Chrome and Firefox extension packages are built together in a single automated step.

## Notes

- This release removes the `activeTab` permission — Starshelf only needs `storage` access to function.

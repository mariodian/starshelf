# v0.1.0

Starshelf now categorizes all your starred repos in one click — no more going through them one by one. This release brings batch categorization, a smoother popup layout, and backend fixes for faster, more reliable bulk updates.

## Highlights

- **Batch categorize** — Click once in the popup to classify every starred repository at once. Progress is tracked per session so you always know where things stand.
- **Redesigned popup** — The batch action has its own dedicated section for a cleaner, more intuitive experience.

## Fixes

- Bulk list operations now use better concurrency throttling, reducing rate-limit errors when updating many repos at once.

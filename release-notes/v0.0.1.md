# v0.0.1

Starshelf brings AI-powered organization to your GitHub stars. With one click, your starred repositories are automatically categorized so you can find what you need without scrolling through an endless list.

## Highlights

- **Instant AI categorization** — Starshelf groups your starred repos into meaningful categories using your choice of Claude, GPT, or OpenCode
- **Star and organize from any repo page** — The overlay appears right on GitHub, no tab switching needed
- **Full control over your data** — Choose which lists are public or private, and pick your preferred AI provider
- **Lightweight and fast** — Runs as a browser extension with minimal overhead

## Notes

This is the initial release of Starshelf. Starshelf requires a GitHub personal access token and an API key for your chosen AI provider. See the README for setup instructions.

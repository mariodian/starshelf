# v0.1.2

Starshelf now has a tabbed popup with search, making it faster than ever to find and manage your starred repositories.

## Highlights

- **Search your stars** — Quickly filter starred repositories by name, description, or language without scrolling through your entire list.
- **Tabbed popup** — The popup is now organized into tabs for categorize, search, and settings, so each feature is just one click away.
- **Sync categories to GitHub** — Your local category assignments can now be pushed back to GitHub star lists, keeping everything in sync.

## Fixes

- Fixed an issue where the extension configuration key was not assigned correctly.

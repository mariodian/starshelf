# v0.0.4

Starshelf now lets you regenerate categories for your starred repos with a single click — no need to clear assignments first. This release also includes under-the-hood improvements that make the extension leaner and easier to maintain.

## Highlights

- **One-click regeneration** — A new button in the popup re-categorizes all your starred repos without removing existing category assignments
- **Leaner codebase** — Internal refactoring removes unused code and organizes remaining logic into clear, focused modules
